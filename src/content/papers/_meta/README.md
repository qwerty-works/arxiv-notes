# Papers

Each paper is a markdown file with frontmatter matching `src/content/config.ts`.

Slug convention: `<arxivId>/<short-slug>`.

Template requirements per post:
- Short catchy title (`catchyTitle`)
- Funny subtitle (`funnySubtitle`)
- Body (Markdown)
- Source link (`sourceUrl`, rendered with target=_blank)
- Next/previous links (auto by `publishedAt`)
- Extra for the homepage: `blurb`
